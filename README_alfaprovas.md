# 🧪 AlfaProvas

Sistema completo para criação e correção de provas escolares.

### 🚀 Funcionalidades
- Upload de provas por professores
- Correção automática com QR Code
- Painel de aluno e professor
- Banco de questões

### 🔗 Em desenvolvimento
Demo (em breve): https://alfaprovas.streamlit.app

### 🛠️ Tecnologias
- Python
- Streamlit
- SQLite (versão demo)
- HTML/CSS
